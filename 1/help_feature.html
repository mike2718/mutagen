☆特徴

- 公開鍵暗号方式・データ中継により高い秘匿性を確保しています。
- キーワードツリーで検索を行うことができます。また絞り込み検索を使うことにより、正規表現やファイルのサイズ・日時など、柔軟な条件で検索できます。
- 自動アップデートを行うことができます。公開鍵暗号方式による電子署名により信頼性を確保しています。自動更新について心配な方は、設定で自動更新を無効にしてください。
- ボードと呼ばれる掲示板機能を備えています。
- あらゆる面で既存のP2Pソフトウェアを上回ることを目的に設計されました。未実装の機能も多いのですが、今後の発展にご期待ください。

☆無視機能

検索画面の無視機能は、"検索に引っかかるためだけに作られた無駄なファイル" を無視するための機能です。
名前が明らかに不正なファイルは無視機能で無視し、その他の正常なファイルは検索で絞り込んでください。
正常なファイルに無視機能を使うと、必要なファイルが見えなくなってしまう危険性があります。

☆評価機能

ダウンロードしたファイルに対して、完了リストで評価を行うことができます。
なお、完了リストからファイルを消去するには、かならず評価をする必要があります。
評価には以下の3種類があります。
優良：質が高く、満足したファイルに対して評価してください。
無害：通常のファイルです。
有害：無効なデータやウイルスに感染したデータ、名前と異なるデータに対して評価してください。
コメント付き評価により、コメントを付加することもできます。

無視機能と評価機能を使用することにより、ネットワーク上で有害なファイルの流通を抑制できます。
適切に無視機能・評価機能を使用して下さいますようお願いします。

☆ファイルの説明

"perfect dark.brain" : ダウンロード情報等の重要なデータが格納されています。
"perfect dark.exe" : 実行ファイルです。
"perfect dark.sekai" : 接続先のIPアドレス/ポートが保存されます。
"perfect dark.settings" : perfect dark の設定が保存されます。
"download" フォルダ : ダウンロードしたファイルのフォルダです。
"unity" フォルダ : キャッシュフォルダです。最大値は設定画面で変更できます。
"mutagen" フォルダ : カスタマイズファイルのフォルダです。設定画面にて作成し、作成したファイルを編集することによりカスタマイズすることができます。
"backup" フォルダ : perfect dark の主要なファイルのバックアップが保存されます。

"readme.txt" : このファイルです。perfect dark を実行すると自動的に作成されます。
"history.txt" : 最新の情報や更新履歴が書いてあります。
"license.txt" : perfect dark の使用許諾書です。必ず読み、同意した上で当ソフトをお使いください。
