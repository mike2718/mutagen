次世代P2P软件 perfect dark

<language.txt>

language.txt 是 perfect dark 的翻译文件。通过编辑这个文件，可以更改几乎所有 perfect dark 中显示的文本。
该文本文件的格式为 UTF-16LE（带有BOM）。请使用支持 Unicode 的文本编辑器进行编辑，保存为UTF-16LE（带有BOM）格式。
language.txt 的语法格式如下。

id "10" /* comment */
" information "
" 信息 "

id 表示文本的ID，紧跟着英语翻译文本和日语翻译文本。
如果要将 perfect dark 翻译为除英语和日语之外的语言，可以直接编辑其中的文本，也可以编辑为类似以下的内容（增加了一行德语文本翻译）

id "10"
" information "
" 信息 "
" informationen "

然后，在设置界面，将显示语言更改为 "其他语言"。（推荐方法）

不一定要在 language.txt 中翻译所有项目。
可以更改需要更改的项目，不需要更改的项目也可以将其删除。
