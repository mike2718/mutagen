<language.txt>

language.txt 是 perfect dark 的语言文件。通过编辑这个文件，你可以更改大部分在 perfect dark 中显示的文本。
此文本文件的格式为 UTF-16LE（带BOM）。请使用支持编辑 Unicode 文本的编辑器对其进行编辑，并将其保存为 UTF-16LE（带BOM）格式。
language.txt 文件具有如下格式：

id "10" /* comment */
" information "
" 信息 "

其中，以 id 开头的行表示一段文本的 ID，跟着的2行分别是英语文本和日语文本。
如果要将 perfect dark 翻译为英语和日语之外的语言，你可以直接编辑其中的文本，建议你先如下编辑（新增一行德语文本）

id "10"
" information "
" 信息 "
" informationen "

之后在设置对话框中，将语言更改为 "其他语言"。

不必更改 language.txt 中的所有项。
你可以更改你要更改的项，不需要更改的项可以删除。
